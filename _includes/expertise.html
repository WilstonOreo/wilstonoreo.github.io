About Me


My name is Michael Winkelmann and software is my passion.

Since 2023, I work as an independent software consultant to help customer to succeed with their software solutions. With a background in computer graphics and image process and my diverse skill set encompassing project management to software architecture and a proven track record in developing user interfaces, I bring a holistic approach to software development.

My main expertise is developing user interfaces for embedded and desktop software with the Qt framework. In 2012, I built my first user interface with Qt while studying Computer Graphics at TU Berlin.

For 7 years, I was working for The Qt Company as Principal Software Engineer and Advisor. During my time at Qt, I helped dozens of customers from different industries and was able to gain deep insights into the framework.

Like Qt, I work cross-platform: I have developed software for Linux, Windows, MacOS and micro-controllers and I am not shy to code on other platforms. Besides my Qt background, I also have used VueJS and the Slint framework for implementing user interfaces.
