About Me


My name is Michael Winkelmann and software is my passion. Since 2023, I work as an independent software consultant to help customer to succeed with their software solutions. With a diverse skill set encompassing Human-Machine Interface (HMI) programming, computer graphics, and a proven track record in project management, I bring a holistic approach to software development.

My main expertise is developing user interfaces for embedded and desktop software with the Qt framework. In 2012, I built my first user interface when studying Computer Graphics at TU Berlin.

For 7 years, I was working for the Qt Company as Software Developer and Consultant. During my time at Qt, I helped dozens of customers from various sectors and was able to gain deep insights into the framework. Besides my Qt background, I also have used VueJS and the Slint framework for implementing user interfaces. I have developed software for Linux, Windows, MacOS and micro-controllers and I am not shy to code on other platforms.

I have gathered deep knowledge about 3D graphics, image processing, rapid manufacting and software project management over the years. Feel free to download my CV and references below if you want to know more about my background.
